Hello, welcome to the Flex repo.

Written almost entirely in ActionScript 3 (AS3), Flex is designed to be a 2D/3D vector animation/game engine. It's a lot like Flash, but margainally more secure'
and it doesn't use a browser plugin, it uses the Flex Projector instead. Flex mainly exports in .fls, but the Flex projector natively supports .swf movies.
Yes, Flex has Flash Player 32, but ran in a much, much better sandbox. It's 3D capabilities mainly come from Flash's Stage3D implementation (since most Flex users
would explort as .swf), but for more modern games, Vulkan is recommended. While it will mainly not use a prowser plugin, the Flex Projector plugin is a stub that relies on the Flex Projector. It usually looks for flxproj.exe in Program Files (x86), which is the main Flex Projector executable. For people who don't use Windows, you may need to past flxprojp (Flex Projector Portable) in the same directory as the Flex Plugin.
